# Requirements: Petcare App

**Defined:** 2026-04-15
**Milestone:** v1.1 AI 증상 분석
**Core Value:** 편의성 우선 — 바쁜 현대인도 반려동물 관리 불안감 없이 반려동물과 행복한 일상

## v1.1 Requirements

### AI Symptom Analysis

- [ ] **SYMP-01**: User can capture symptom photo using device camera
- [ ] **SYMP-02**: User can select which pet to analyze
- [ ] **SYMP-03**: System sends photo to AI and receives symptom analysis
- [ ] **SYMP-04**: System displays urgency level (Red/Yellow/Green) with analysis
- [ ] **SYMP-05**: System displays recommended actions based on analysis
- [ ] **SYMP-06**: System displays mandatory medical disclaimer
- [ ] **SYMP-07**: User can save analysis to pet health timeline
- [ ] **SYMP-08**: User can view symptom analysis history

### Future Requirements (Deferred)

- **SYMP-09**: Multi-modal input (photo + text description)
- **SYMP-10**: Voice output for analysis results
- **SYMP-11**: Direct vet booking from analysis results
- **SYMP-12**: Custom CV model for specific condition detection

## Out of Scope

| Feature | Reason |
|---------|--------|
| On-device AI processing | Too resource-intensive, cloud-based AI more accurate |
| Real-time video analysis | Still image sufficient for symptom analysis |
| Regulatory medical device claim | Informational only, not diagnostic |
