Create server/src/utils/dietCalculator.ts:

1. **CalorieTarget calculation**:
   - Based on pet weight and species
   - Formula: weight(lb) * species_multiplier
   - Species multipliers: dog=30kcal/lb, cat=20kcal/lb, bird=10kcal/lb, rabbit=15kcal/lb

2. **Macronutrient estimation** (per CONTEXT.md):
   - Default estimates: protein=25%, fat=15%, carbs=60%
   - If not provided, estimate from calories and defaults
   - protein/carb = 4kcal/g, fat = 9kcal/g

Create server/src/controllers/diet.controller.ts:

1. **Endpoints**:
   - GET /api/pets/:id/diets - List diet entries (by date range)
   - POST /api/pets/:id/diets - Log meal (DIET-01)
   - PUT /api/pets/:id/diets/:dietId - Update meal
   - DELETE /api/pets/:id/diets/:dietId - Delete meal
   - GET /api/pets/:id/diets/stats - Get daily/weekly summary (DIET-02, DIET-03, DIET-04)

2. **POST /api/pets/:id/diets flow**:
   - Validate input with zod (foodName 1-200 chars, amountGrams 1-10000, calories 1-10000)
   - Auto-calculate macros if not provided
   - Create diet record
   - Return created diet entry

3. **GET /api/pets/:id/diets/stats flow**:
   - Fetch today's entries for dailyCalories
   - Calculate caloriesConsumed vs calorieTarget (DIET-02)
   - Aggregate macros (DIET-03)
   - Fetch 7-day data for weekly trends (DIET-04)
   - Return DietStats

Create server/src/services/diet.service.ts:

1. **DietService class**:
   - `async logDiet(petId: string, data: CreateDietRequest): Promise<Diet>`
   - `async getDiets(petId: string, days?: number): Promise<Diet[]>`
   - `async getDietStats(petId: string): Promise<DietStats>`
   - `async calculateCalorieTarget(species: string, weightKg: number): number`
   - `async estimateMacros(foodName: string, amountGrams: number, calories: number): MacroEstimate`

Create server/src/routes/diet.routes.ts:
