Create server/src/controllers/activity.controller.ts:

1. **Endpoints**:
   - GET /api/pets/:id/activities - List activities (by date range, default last 30 days)
   - POST /api/pets/:id/activities - Log activity (ACT-01, ACT-02)
   - PUT /api/pets/:id/activities/:activityId - Update activity
   - DELETE /api/pets/:id/activities/:activityId - Delete activity
   - GET /api/pets/:id/activities/stats - Get 7-day summary + chart data (ACT-04)

2. **POST /api/pets/:id/activities flow**:
   - Validate input with zod (durationMinutes: 1-1440, steps: optional, date: ISO)
   - Create activity record
   - Return created activity

3. **GET /api/pets/:id/activities/stats flow**:
   - Fetch activities from last 7 days
   - Group by day, sum durationMinutes and steps
   - Calculate totalMinutes, totalSteps, averageDaily
   - Fetch pet data for species/weight to calculate goalMinutes
   - Calculate goalProgress percentage
   - Generate chartData array (7 objects, one per day)
   - Return ActivityStats

Create server/src/services/activity.service.ts:

   - `async logActivity(petId: string, data: CreateActivityRequest): Promise<Activity>`

   - `async getActivityStats(petId: string): Promise<ActivityStats>`
   - `async calculateDailyGoal(species: string, weightKg: number): number`
   - Uses date-fns for date calculations
