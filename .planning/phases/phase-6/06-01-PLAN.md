## Tasks

### Task 1: Push Notification Service
- Install @react-native-firebase/messaging
- Create NotificationService with:
  - requestPermission() for iOS/Android
  - getToken() for FCM
  - onMessage() handler for foreground notifications
- Create vaccination reminder notifications when due

### Task 2: Notification History
- Create notificationStore with history array
- Store notification history in AsyncStorage
- Create NotificationHistoryScreen
- Display notification history with timestamp, title, body

### Task 3: Offline Caching Service
- Create offlineCacheService with:
  - cachePetProfiles() - cache selected pet data
  - cacheMedicalRecords() - cache medical records
  - getCachedPets() - retrieve cached pets
  - getCachedRecords() - retrieve cached records
  - clearCache() - clear all cached data
- Use AsyncStorage with cache-first strategy

### Task 4: Error Boundary
- Create ErrorBoundary component
- Wrap root App with ErrorBoundary
- Fallback UI with friendly message
- Error reporting/logging hook

### Task 5: EAS Build Configuration
- Create eas.json with:
  - build profiles: development, preview, production
  - iOS: IPA export for simulator/device
  - Android: AAB export
- Configure app.json with build settings
