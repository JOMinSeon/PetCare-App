## Tasks

### Task 1: User Emergency Contact
- Add emergencyPhone field to User model (or use existing phone)
- Update user API to include emergency contact
- Display in pet ID card

### Task 2: QR Code Generation
- Install react-native-qrcode-svg and react-native-svg
- Create QRCodeCard component with pet info encoded as JSON
- JSON payload: { petId, petName, species, ownerContact }
- Display QR in pet profile and separate ID screen

### Task 3: Digital ID Card Screen
- Create DigitalIDScreen showing:
  - Pet photo/avatar
  - Pet name, species, breed, weight, age
  - QR code
  - Emergency contact button
- Navigation from pet profile or dashboard

### Task 4: QR Scanner Screen (for vets/shelters)
- Install react-native-vision-camera
- Create QRScannerScreen with camera view
- On successful scan: display pet info in read-only view
- Emergency info display for shelter staff
