## Tasks

### Task 1: Server Medical Record API
- Create MedicalRecordController with CRUD endpoints
- Endpoints: GET /api/pets/:id/medical-records, POST, PUT, DELETE
- Filter by type: vaccinations, checkups
- Add next_due_date for vaccination reminders

### Task 2: Mobile Medical Record Service & Store
- Create medicalRecord.service.ts
- Create medicalRecordStore.ts
- CRUD actions: fetchRecords, createRecord, updateRecord, deleteRecord

### Task 3: Medical Record Screens
- Create MedicalRecordListScreen (grouped by type)
- Create AddMedicalRecordScreen (form with type selector)
- Create MedicalRecordDetailScreen (view/edit)

### Task 4: Medical Record Cards
- Create MedicalRecordCard component
- Grouped display with type icon (💉 vaccination, 🏥 checkup)
- Show next due date if applicable
