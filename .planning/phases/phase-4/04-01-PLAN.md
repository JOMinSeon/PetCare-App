## Tasks

### Task 1: Server-side Service Data Model & API
- Verify Service model exists in Prisma schema (from Phase 1)
- Create mock service data seeder for development
- Create `GET /api/services` endpoint with location-based query
- Create `GET /api/services/nearby` with lat/lng/radius params

### Task 2: Mobile Map Setup
- Install `react-native-maps` package
- Configure Google Maps API key (via environment variable)
- Create MapScreen with full-screen map view
- Display user location as initial region

### Task 3: Service Markers
- Create ServiceMarker component
- Display markers on map for each service
- Marker icon color by service type (vet=red, store=blue, etc.)
- Tap marker to show service card

### Task 4: Filter Chips
- Create FilterChip component (horizontal scroll)
- Service types: All, Vet, Pet Store, Groomer, Emergency
- Create filter state and apply to map markers
- 24-hour toggle filter switch

### Task 5: Service Card (Bottom Sheet)
- Create ServiceCard component
- Display: name, type, distance, rating, open/closed status
- Call button (phone icon → tel: link)
- Navigate button (map icon → Google Maps/Apple Maps)
- Bottom sheet animation on marker tap
