## Phase Details

### Phase 1: Foundation & Health Management

**Goal:** Users can create accounts, manage pet profiles, record health information, and receive timely reminders

**Depends on:** None (first phase)

**Requirements:** AUTH-01, AUTH-02, AUTH-03, PET-01, PET-02, PET-03, HEAL-01, HEAL-02, HEAL-03, HEAL-04, NOTI-01, NOTI-02, NOTI-03, NOTI-04

**Success Criteria** (what must be TRUE):
1. User can sign up with email/password and log in to access the app
2. User can register one or more pets with name, species, breed, birth date, and photo
3. User can edit pet profile information at any time
4. User can record vaccination history with vaccine name, date, and next due date
5. User can record medication schedules with medicine name, dosage, frequency, and dates
6. User can record medical examination history with clinic, date, diagnosis, and notes
7. User can view a chronological timeline of all health records for each pet
8. User receives push notification reminders for scheduled medications
9. User receives push notification reminders when vaccination due date approaches
10. User receives push notification reminders for scheduled walk times
11. User can create custom reminders for any health-related task

**Plans:** 3/4 plans complete
- 01-01-PLAN.md — Authentication (signup, login, JWT)
- 01-02-PLAN.md — Pet profiles (CRUD, model)
- 01-03-PLAN.md — Health records (vaccination, medication, examination, timeline)
- 01-04-PLAN.md — Notifications and reminders (push, custom reminders)

### Phase 2: Community & Analytics

**Goal:** Users can share pet moments with the community and view health trends

**Depends on:** Phase 1

**Requirements:** COMM-01, COMM-02, COMM-03, COMM-04, ANLY-01, ANLY-02, ANLY-03

**Success Criteria** (what must be TRUE):
1. User can create posts with photos and captions to share pet moments
2. User can like posts from other community members
3. User can comment on posts to engage with the community
4. User can browse a feed of recent posts from all community members
5. User can view a weight trend chart for each pet
6. User can view an activity level trend chart for each pet
7. User receives basic health insights based on recorded pet data

**Plans:** 2/2 plans complete
- 02-01-PLAN.md — Community posts (feed, create, like, comment)
- 02-02-PLAN.md — Health analytics (weight/activity charts, insights)
**UI hint:** yes

### Phase 3: Service Discovery

**Goal:** Users can find nearby veterinary clinics, book appointments, and leave reviews

**Depends on:** Phase 1

**Requirements:** SERV-01, SERV-02, SERV-03, SERV-04

**Success Criteria** (what must be TRUE):
1. User can search for veterinary clinics near their location
2. User can view clinic details including address, phone number, and operating hours
3. User can select a date and time to book an appointment at a clinic
4. User can write and submit a review for clinics they have visited

**Plans:** 3/3 plans complete
- 03-01-PLAN.md — Clinic search & map (Kakao Maps, list, detail screen) ✅ COMPLETE
- 03-02-PLAN.md — Booking system (date/time picker, booking history) ✅ COMPLETE
- 03-03-PLAN.md — Review system (star rating, text review, submission) ✅ COMPLETE
**UI hint:** yes
