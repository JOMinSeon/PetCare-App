# Requirements: Petcare App

**Defined:** 2026-04-15
**Core Value:** 편의성 우선 — 바쁜 현대인도 반려동물 관리 불안감 없이 반려동물과 행복한 일상

## v1 Requirements

### Pet Profile

- [x] **PET-01**: User can register pet profile (name, species, breed, birth date, photo)
- [x] **PET-02**: User can edit pet profile information
- [x] **PET-03**: User can register multiple pets

### Health Record

- [x] **HEAL-01**: User can record vaccination history (vaccine name, date, next due date)
- [x] **HEAL-02**: User can record medication schedule (medicine name, dosage, frequency, start/end date)
- [x] **HEAL-03**: User can record medical examination history (clinic, date, diagnosis, notes)
- [x] **HEAL-04**: User can view health history timeline

### Notifications & Reminders

- [x] **NOTI-01**: System sends medication reminder at scheduled time (push notification)
- [x] **NOTI-02**: System sends vaccination reminder when due date approaches
- [x] **NOTI-03**: System sends walk schedule reminder
- [x] **NOTI-04**: User can set custom reminder for any health-related task

### Health Analytics

- [x] **ANLY-01**: User can view weight trend chart
- [x] **ANLY-02**: User can view activity level trend
- [x] **ANLY-03**: System provides basic health insights based on recorded data

### Community

- [x] **COMM-01**: User can post photos with caption (pet activity, moments)
- [x] **COMM-02**: User can like posts
- [x] **COMM-03**: User can comment on posts
- [x] **COMM-04**: User can view community feed (recent posts)

### Service Discovery

- [x] **SERV-01**: User can search nearby veterinary clinics
- [x] **SERV-02**: User can view clinic information (address, phone, hours)
- [x] **SERV-03**: User can book appointment (select date/time)
- [x] **SERV-04**: User can write reviews for clinics

### Authentication

- [x] **AUTH-01**: User can sign up with email and password
- [x] **AUTH-02**: User can log in with email and password
- [x] **AUTH-03**: User session persists across app restart

## v2 Requirements

### Service

- **SERV-05**: Integration with pet pharmacy for medication ordering
- **SERV-06**: Pet hotel/grooming reservation
- **SERV-07**: Product recommendation based on pet profile

### Community

- **COMM-05**: Follow/unfollow other users
- **COMM-06**: Direct messaging between users

### Analytics

- **ANLY-04**: Advanced health trend analysis with veterinarian export

## Out of Scope

| Feature | Reason |
|---------|--------|
| Real-time chat | Community is async photo-sharing focus |
| Video streaming | Image/photo sharing only for v1 |
| Expert dashboard | General users, not industry professionals |
| Multi-language | Korean-only for v1 |
